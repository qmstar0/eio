# eio

</div>

[English](README.md)|中文

## 简介

`eio` 是一个提供事件驱动规范的模块。经过多次迭代后，它放弃了提供具体的事件，而是转向提供相对抽象的接口，并为消息序列化提供了方法。

## 特性

- 提供事件驱动规范
- 转向相对抽象的接口设计
- 提供消息序列化方法

## 安装

```bash
go get -u github.com/qmstar0/eio
```
